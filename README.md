# 🔢 Mini Processor for BCD Arithmetic Operations

A digital electronics project designed to perform BCD arithmetic operations including addition, subtraction, and multiplication using digital logic circuits and Proteus simulation.

---

# 📌 Project Overview

This project was developed for the Digital Electronics module at SLIIT.  

The system accepts two 1-digit BCD numbers as inputs and performs arithmetic operations based on control signals.

The processor was designed and simulated using Proteus to demonstrate practical digital circuit implementation.

---

# ⚙️ Supported Operations

- ➕ Addition (00)
- ➖ Subtraction (01)
- ✖️ Multiplication (10)

---

# 🔢 Output Features

- 2-digit BCD output (00–99)
- Overflow detection for values greater than 99

---

# 🧩 Main Components Used

- 🔹 BCD Adder
- 🔹 BCD Subtractor
- 🔹 BCD Multiplier
- 🔹 Multiplexer (MUX)
- 🔹 Registers
- 🔹 Control Logic Circuits

---

# 🛠️ Tools & Technologies

- 💻 Proteus Simulator
- 🔌 Digital Logic Design
- ⚡ BCD Arithmetic Circuits
